Description of above menu

Sign in

This is where a user will enter his/her address in order to check his/her permission to access the free mentor system. Once registered and enter correct credntials he/she will be allowed to access following pages. In case he/She can't access other pages he will be asked to register or enter correct credentials.

Sign up

This is where a user will register in order to be allowed to access other pagers passing to sign in.

View mentors

This is where a user will pass to see all mentors in the free mentor system and decide who he/she will initiate a mentorship session

Specific mentor

This is where a user will pass to see all mentors in the free mentor system and decide who he/she will initiate a mentorship session

session with mentor

This is where a user will use to initiate a conversation with a mentorship.
